# console.lol - Because I can

Console.log, but better:

## Install

```
npm install console.lol
```

## Usage

```javascript
console.lol("Initializing user", user.id);
// (╯°□°)╯︵ ┻━┻  Initializing user 377
```

```javascript
console.lol("GET", url);
// ¯\_(ツ)_/¯ GET /api/users
```

```javascript
console.rofl("Hello how are you?")
